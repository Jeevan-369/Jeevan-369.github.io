Recent Projects

Project 1

Brain Computer Interface for Emergency Commands

This focuses on utilizing Brain-Computer Interface (BCI) technology, specifically the NeuroSky Mindlink Neuro Sensor, to enable speech communication for individuals with severe muscular illnesses. By processing raw EEG data and extracting attention levels, basic commands are generated, enhancing communication for conditions like ALS and stroke-related paralysis, fostering independence and improved quality of life.

Project 2

Text Encryption and Decryption app using various algorithms

This mobile application is built using Java, employing Arduino software for hardware communication. Users input plain text, choose encryption algorithms (e.g., AES, DES, MD5), and provide a key. The app encrypts the text, generating cipher text. Users send this via any messaging app, and recipients decrypt using the chosen algorithm and the sender's key.

Project 3

Voter ID management system using B+ tree indexing technique

The Voter ID Management System, developed in C++ features a user-friendly menu-driven interface using local file systems. It stores data on disk, allowing users to view, add, search, delete, and modify voter records. The B+ tree indexing technique optimizes record access by maintaining byte offsets in the original file.

Project 4

Covid Bed Slot Booking System using python flask and MySQL

The COVID-19 bed slot booking system, developed with Python Flask, MySQL, and Bootstrap, offers a user-friendly solution for patients to reserve hospital beds. Utilizing XAMPP for efficient database management, this platform streamlines COVID care, providing a robust and accessible tool for securing vital medical resources during the ongoing pandemic.
